## 2. 环境准备

### 2.1 安装依赖

**Python 环境（后端）**

```bash

conda create -n cccc python=3.12
conda activate cccc

# 安装依赖
cd f:\repos\employment-backend
pip install -r requirements.txt
```

**Node.js 环境（前端）**

```bash
# 使用 pnpm
cd f:\repos\art-design-pro
pnpm install
```

### 2.2 启动 MySQL

```bash
# Windows 服务模式（已安装 MySQL）
sc start MySQL80

# 或使用 Docker
docker run -d --name mysql -p 3306:3306 \
  -e MYSQL_ROOT_PASSWORD=你的密码 \
  mysql:8
```

### 2.3 启动 Redis（可选，AI 功能需要）

```bash
# Windows（安装 Redis 或使用 Docker）
docker run -d --name redis -p 6379:6379 redis:7-alpine

# 或下载 Redis Windows 版：https://github.com/microsoftarchive/redis/releases
```

## 3. 启动后端

### 3.1 配置文件

创建 `.env` 文件：

```bash
cd f:\repos\employment-backend
copy .env.example .env
```

编辑 `.env`：

```env
DATABASE_URL=mysql+aiomysql://root:你的密码@localhost:3306/employment_db
REDIS_URL=redis://localhost:6379/0
JWT_SECRET_KEY=your-secret-key-here-change-in-production
JWT_ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=15
REFRESH_TOKEN_EXPIRE_DAYS=7
APP_ENV=development
```

### 3.2 初始化数据库

```bash
conda activate cccc
cd f:\repos\employment-backend

# 运行数据库迁移
alembic upgrade head

# 创建管理员账号
python -m app.cli create-admin admin Admin123!
```

### 3.3 启动服务

```bash
# 激活环境
conda activate cccc

# 启动后端（开发模式）
cd f:\repos\employment-backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 5174
```

或直接双击运行 `start.bat`（位于 `f:\repos\employment-backend\start.bat`）

后端启动后访问：
- API 文档：http://localhost:5174/docs
- 健康检查：http://localhost:5174/api/v1/health

### 3.4 测试登录

```bash
curl -X POST "http://localhost:5174/api/v1/auth/login" \
  -H "Content-Type: application/json" \
  -d "{\"username\":\"admin\",\"password\":\"Admin123!\",\"role\":\"system_admin\"}"
```

## 5. 数据库维护

### 5.1 数据库迁移

```bash
conda activate cccc
cd f:\repos\employment-backend

# 查看当前版本
alembic current

# 查看迁移历史
alembic history

# 升级到最新
alembic upgrade head

# 降级一个版本
alembic downgrade -1

# 重新生成迁移（模型修改后）
alembic revision --autogenerate -m "描述"
alembic upgrade head
```

### 5.2 重置数据库

```bash
# 删除并重建数据库
mysql -u root -p -e "DROP DATABASE IF EXISTS employment_db; CREATE DATABASE employment_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

# 重新迁移
alembic upgrade head

# 重新创建管理员
python -m app.cli create-admin admin Admin123!
```

### 5.3 手动操作数据库

```bash
mysql -u root -p123456 employment_db

# 查看所有表
SHOW TABLES;

# 查看账号
SELECT account_id, username, role, status FROM accounts;

# 删除账号
DELETE FROM accounts WHERE username = 'admin';
```

### 5.4 CLI 命令

```bash
python -m app.cli <command>

# 可用命令：
# init-db                     - 初始化数据库表
# create-admin <user> <pass>  - 创建管理员
# seed-data                   - 创建测试数据
# verify-company <id> <action> - 审核企业
# unlock-account <username>    - 解锁账号
```

## 6. 常见问题

### 6.1 Redis 连接失败

```
WARNING: Redis 连接失败: Error 10061
```

**解决**：启动 Redis 服务，或忽略此警告（基础功能不受影响）

### 6.2 MySQL 连接失败

```
ModuleNotFoundError: No module named 'aiomysql'
```

**解决**：
```bash
pip install aiomysql pymysql
```

### 6.3 端口被占用

```
Error: That port is already in use
```

**解决**：
```bash
# 查找占用端口的进程
netstat -ano | findstr :5174

# 结束进程（PID 为进程号）
taskkill /PID <进程号> /F
```

### 6.4 密码哈希错误

```
AttributeError: module 'bcrypt' has no attribute '__about__'
```

**解决**：
```bash
pip install --upgrade passlib bcrypt
```

### 6.5 迁移失败 - 外键约束

```
OperationalError: (1824, "Failed to open the referenced table")
```

**解决**：重置数据库
```bash
mysql -u root -p -e "DROP DATABASE IF EXISTS employment_db; CREATE DATABASE employment_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
alembic upgrade head
```

### 6.6 前端无法访问后端 API

检查 Vite 代理配置 `.env.development`：
```env
VITE_API_PROXY_URL=http://localhost:5174
```

确保后端运行在 5174 端口。
