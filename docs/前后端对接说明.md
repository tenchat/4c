# 前后端对接说明文档

本文档记录前端页面与后端API的对接情况，包括字段映射、接口调用和已完成的修改。

## 1. 修复的问题

### 1.1 API URL 统一前缀

**问题**: 部分 API 路径缺少 `/api` 前缀

**修改文件**: `src/api/student.ts`

| API 函数 | 修改前 | 修改后 |
|---------|--------|--------|
| `fetchStudentDashboard` | `/v1/student/dashboard` | `/api/v1/student/dashboard` |
| `fetchStudentProfile` | `/api/v1/student/profile` | `/api/v1/student/profile` (不变) |
| `fetchStudentJobs` | `/v1/student/jobs` | `/api/v1/student/jobs` |
| `applyForJob` | `/api/v1/student/jobs/${jobId}/apply` | `/api/v1/student/jobs/${jobId}/apply` (不变) |

## 8. 注意事项

1. **所有 API 请求都需要在 Header 中携带 `Authorization: Bearer {token}`**
2. **企业端接口需要先通过 `company_id` 关联账号**
3. **学校端接口需要先通过 `university_id` 关联账号**
4. **后端返回的时间格式为 ISO 8601 格式，前端需要自行格式化显示**
